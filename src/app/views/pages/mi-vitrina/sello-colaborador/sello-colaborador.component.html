¿Cómo usar el sello?
  1. Selecciona el tamaño que mejor se adapte al diseño de tu sitio web
  2. Haz clic en "Copiar código HTML" del sello que prefieras
  3. Pega el código en el HTML de tu página web donde desees mostrar el sello
  4. El sello será visible y enlazará directamente a tu perfil en Bodas Planner
